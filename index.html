ibuffer-git by jrockway

Git integration for Emacs' Ibuffer

Dependencies

Emacs 23.1

Install

Copy ibuffer-git.el to a convenient location, (require 'ibuffer-git), and add git-status or git-status-mini to ibuffer-formats.
